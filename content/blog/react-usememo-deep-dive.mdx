## Why useMemo exists

React re-renders often. `useMemo` caches computed values.

![Preview](/cv-preview.png)

## The real cost

### Memory vs CPU

```js
import { useMemo } from "react";

export function Example({ data }) {
  const expensive = useMemo(() => heavyCompute(data), [data]);
  return <div>{expensive}</div>;
}
```

## Takeaways

- Prefer readability first
- Measure before optimizing
